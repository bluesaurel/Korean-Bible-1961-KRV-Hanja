# 출처 변이 및 독립 자료 확인 시트

작성일: 2026-06-29

이 문서는 위키문헌 공개본이 중요한 단서를 주지만, 공개 확정 전에 실물본
또는 독립 자료 확인이 필요한 항목을 모은 것입니다.

## 확인 원칙

- 독립 자료 확인 전에는 실제 2차 적용 manifest에 넣지 않는다.
- 공개본 raw 이상 가능성이 있는 항목은 한 자료만 보고 보정하지 않는다.
- 본문 누락 보정과 맞춤법/표기 차이는 한 번에 섞어 승인하지 않는다.

## 독립 자료 확인 요청 메모

- 실물 또는 스캔본의 판본명, 발행처, 발행연도, 쪽/열/절 위치를 기록한다.
- 온라인 자료는 URL, 조회일, 줄 번호 또는 화면 캡처 근거를 함께 기록한다.
- 확인 결과는 `manualDecision`, `approvedBy`, `approvedDate`를 채운 승인 manifest로만 적용한다.
