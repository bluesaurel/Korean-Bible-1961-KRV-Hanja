# 보존 후보 설명문

작성일: 2026-06-29

이 문서는 한글 기준 KRV와 다르지만 외부 국한문 공개본이 로컬 독법, 절 경계,
또는 표기를 지지하는 항목을 공개 시 어떻게 설명할지 준비하는 문서입니다.

## 원칙 초안

- 보존 후보는 오류 확정 항목이 아니다.
- 공개본에는 로컬 국한문 독법을 유지하되, 기준 KRV와 다른 이유를 검수 계보에 기록한다.
- 위키문헌 공개본만으로 최종 확정하지 않고, 가능하면 실물본 또는 독립 자료 확인을 추가한다.
- 보존 결정을 내리더라도 원본 TXT와 raw JSON은 수정하지 않는다.
